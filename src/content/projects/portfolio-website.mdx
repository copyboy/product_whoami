# Portfolio Website

A clean, modern portfolio website built with Next.js and Tailwind CSS. The site is designed to showcase my projects, skills, and professional experience in an elegant and responsive layout.

## Features

- **Responsive Design**: Looks great on all devices from mobile to desktop
- **Dark Mode**: Automatic and manual dark mode support
- **Fast Loading**: Built with performance in mind
- **SEO Optimized**: Meta tags and structured data for better search visibility
- **Contact Form**: Integrated contact form with email notifications
- **Analytics**: Google Analytics integration
- **Animations**: Subtle animations enhance the user experience
- **Blog Section**: Share thoughts and tutorials with an integrated blog

## Technologies Used

- **Next.js**: For server-side rendering and optimal performance
- **Tailwind CSS**: For rapid UI development
- **TypeScript**: For type safety and better developer experience
- **Framer Motion**: For animations
- **React Hook Form**: For form handling
- **Sendgrid**: For email functionality

## Project Structure

The project follows a modular structure for easy maintenance:

```
/
├── components/
│   ├── common/
│   ├── layout/
│   └── sections/
├── pages/
│   ├── api/
│   ├── blog/
│   └── projects/
├── public/
│   ├── images/
│   └── assets/
└── styles/
    └── globals.css
```

## Deployment

The site is deployed on Vercel, which provides automatic deployments, preview environments, and analytics.
